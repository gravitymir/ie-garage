Drop file here, or click to pick one
Any .txt / .csv from your Android scanner app — one code per line.
Or paste scans directly:
Format: one scan per line. Duplicates are counted (3 lines with the same code = quantity 3). Or use code,count (or tab / space separated) if your scanner already sums them, e.g. 6959655449112,3. Lines starting with # are ignored.
Strict: items you didn't scan are set to 0 (you counted everything, so absent = zero on shelf). Partial: unmatched items stay untouched.
